# Screenshots Directory

This directory contains screenshots of the Disease Prediction Model application.

## Required Screenshots

To complete the README, please add the following screenshots:

1. **prediction-interface.png** - Screenshot of the Disease Prediction interface showing:
   - Input fields for symptoms and state
   - Prediction results with scores
   - Tree voting breakdown

2. **interactive-map.png** - Screenshot of the Interactive Map showing:
   - Disease clusters on the India map
   - Cluster markers and information
   - Legend panel

## How to Add Screenshots

1. Take screenshots of your running application
2. Save them with the exact names:
   - `prediction-interface.png`
   - `interactive-map.png`
3. Place them in this `screenshots/` directory
4. Commit and push to GitHub:
   ```bash
   git add screenshots/*.png
   git commit -m "Add application screenshots"
   git push
   ```

## Screenshot Tips

- Use PNG format for best quality
- Recommended size: 1200x700 pixels or larger
- Make sure the interface is clearly visible
- Capture the full application window
